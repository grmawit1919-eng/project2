Library

Since its establishment in 1993, the Mekelle University Library System has grown remarkably. What began as a single classroom with donated and purchased books has transformed into a robust academic resource hub. With the official founding of Mekelle University in 2000, the library system expanded to include 11 branch libraries, two main library buildings, and a comprehensive digital library network.
